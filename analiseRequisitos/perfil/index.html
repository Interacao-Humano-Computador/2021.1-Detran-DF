    1. Perfil do Usuário

    A definição de um perfil de usuário é extremamente importante para o projeto pois somente a partir deste é possível conhecer os usuários que utilizarão o sistema, quais seus interesses e quais suas características (como idade, experiência e quais as atividades que irão realizar utilizando o projeto).

    Tendo consciência de tal importância o grupo decidiu utilizar de questionários para realizar o levantamento do perfil. Após isso criaram-se as personas, baseado num resultado bastante homogêneo já que o nosso projeto é baseado no site Detran DF, uma plataforma utilizada pelos mais diversos tipos pessoas das mais diversas idades.
